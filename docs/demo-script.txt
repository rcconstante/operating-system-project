LaSallianOS Demo Script
=======================

1. Open the terminal in the project root.
2. Build the ISO:
   make
3. Run the OS:
   make run
4. Wait for the boot banner:
   LaSallianOS v1.0
   System initialization complete.
5. Type:
   help
6. Explain that the shell reads keyboard input and dispatches command handlers.
7. Type:
   about
8. Explain the modules: boot, kernel, screen, keyboard, shell, process, memory.
9. Type:
   memory
10. Explain that this is a memory management simulation.
11. Type:
   process
12. Explain that this is a process table and scheduler simulation.
13. Type:
   clear
14. Explain that VGA text mode output is being cleared directly.
15. Type:
   shutdown
16. Explain that the CPU is halted and QEMU can be closed.
